添加角色和表情

 

在这一部分，我们将教你如何在视觉小说制作器中添加角色！

 

 

创建文件夹

  • 点击 数据库标签页
  • 选择 Collections -> Characters
  • 让我们创建一些文件夹！就像之前提到的那样。 制作角色列表, 我们将创建一个按重要性顺序分隔所有角色的列表。只需点击 右键 -> New Folder.
  • 点击 New Folder 重命名它。
  • 根据你正在创作的故事类型，你可能只需要几个文件夹。为了简单起见，我只创建了三个文件夹。

 

创建角色

  • 现在，选择一个文件夹，比如主角，然后 右键 -> New Record.
  • 点击 New Record 并点击“名称”来重命名，就像这样：
  • 恭喜！我们创建了一个角色！现在填写相关的角色信息。让我们以小红帽为例：

 

创建角色表情

  • 选择 Collections -> Character Expressions
  • 根据你的视觉小说长度，有不同的导入资源的方法。以下是两种方法：

 

使用表情导入器

如果你希望使用自动导入工具。只需阅读表情导入器页面。

 

手动添加表情

如果你想手动添加表情，这个过程与添加角色类似。

 

  • 阅读导入资源页面。
  • 类似于 Characters 只需创建文件夹将它们彼此分开。
  • 选择一个文件夹，例如主角，然后 Right Click -> New Record.
  • 点击 New Record 并点击“名称”来重命名它。
  • 您的数据库应该看起来像这样：
  • 只需点击添加，然后选择您想要为特定情绪显示的图像文件。如果操作正确，数据库应该像这样显示您的角色表情：

  • 继续为所有角色执行这些步骤！这样您将设置好所有角色的表情！

 

额外功能：为您的角色设置缩略图

作为一个巧妙的方式来显示当前场景中的角色，请返回 Characters 页面。

  • 点击所需的角色。
  • 点击 默认表情 并选择您希望角色默认显示的表情
  • 现在每次将该角色添加到场景中时，您将看到类似于这样的效果！
