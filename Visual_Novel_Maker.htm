什么是视觉小说制作器？

Visual Novel Maker是一个专攻于制作视觉小说和其子类型的引擎。 它可以让用户创造交互或者非交互式的故事，并发布到PC端以及移动端。这种易于上手的编辑器，可以让你在没有编程知识的前提下相对轻松地创作视觉小说。视觉小说编辑器是由André Radomski所开发的。

 

主要特点

Visual Novel Maker具有强大的用户友好界面，同时为程序员们保留了灵活性和高级功能。

 

支持多平台

Create games in Windows, MacOSX and Linux Ubuntu!

 

易于理解的场景编辑器

场景编辑器可以让你更轻松的为游戏创造场景。

 

精简的数据库（Streamlined Database）

这种数据库允许用户自由地修改数据，而不受特别的限制。它也允许用户在文件夹内轻松排序数据。

 

高级脚本和插件系统

脚本语言支持CoffeeScript或者JavaScript。你可以很容易地定制你的游戏外观或改变整个游戏系统。如果你需要更多的功能，你甚至借助强大的插件系统为编辑器添加新的功能。

 

Language Configuration

你想把游戏翻译成其他语言的吗？使用这个特性，你可以很轻松办到这点。

 

Multiple Game Package Options

You can build games for multiple OSes and Devices

  • HTML5 / Web
  • Windows 32/64 Bit
  • Linux Ubuntu 32/64 Bit
  • OS X 32/64 Bit
  • SteamOS
  • Android
  • iOS

 

最低配置要求

使用本Visual Novel Maker所需的最低配置如下：

 

操作系统（OS）

 Windows 7/8.1/10 (64bit), Mac OS X 10.10 (64-Bit), Linux Ubuntu (64-Bit)

处理器（Processor）

 Intel Core 2 Duo or better

显卡（Graphics）

 DirectX / OpenGL capable GPU

显示器（Display）

 1280x768 or better Display

内存（Storage）

 3 GB available space

Additional Notes

 OpenGL hardware driver support required for OpenGL accelerated rendering..
